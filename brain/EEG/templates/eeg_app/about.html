About NeuroMood

NeuroMood is an experimental web application that uses EEG signal processing and AI to estimate emotional states (valence, arousal, dominance, discrete emotions) from brainwave data.

Important Disclaimer

This tool is not a medical device, diagnostic tool, or substitute for professional mental health care. Results are estimates only and should not be used for clinical decisions.

How it works (high-level)

  • Upload cleaned EEG data (.csv / .txt / .npy)
  • Preprocessed features → deep learning model predicts emotion dimensions
  • RAG-enhanced LLM generates explanation, action steps & follow-up questions
  • Chat interface allows you to ask more specific questions about your session

Technology stack

  • Django (backend)
  • Tailwind CSS + HTMX + Alpine.js (frontend)
  • Chart.js (visualizations)
  • Custom emotion inference model + LLM agent (RAG)

Built as an experimental side project • 2024–2025
